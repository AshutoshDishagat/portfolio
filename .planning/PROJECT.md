# Full-Stack Portfolio (Next.js & MongoDB)

A modern, professional portfolio website designed for a Full-Stack Web Developer. The project aims for a "premium feel" to be used for freelancing platforms (Upwork/Fiverr) and high-end job applications. 

Drawing inspiration from the [codebucks27/Next.js-Developer-Portfolio-Starter-Code](https://github.com/codebucks27/Next.js-Developer-Portfolio-Starter-Code) design, it features bold typography, minimalist high-contrast (Dark/Light) aesthetics, and interactive animations.

## Core Value

To establish a high-trust, elite professional brand through a high-performance, visually stunning portfolio that proves both Engineering (MERN/Next.js) and Design (UI/UX) capabilities.

## Context

- **Audience**: Hiring managers, technical leads, and freelance clients.
- **Vibe**: Professional, sleek, and high-performance.
- **Reference**: Clean minimalist design with interactive Framer Motion components (orbiting skills, grid-based project cards).
